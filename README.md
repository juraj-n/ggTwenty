# ggTwenty

## Popis aplikácie

Aplikácia je webová aplikácia vyvíjaná v rámci semestrálneho projektu.
Zdrojový kód aplikácie je dostupný na platforme [GitHub](https://github.com/juraj-n/ggTwenty.git)

---

## Požiadavky

Pre správne spustenie aplikácie je potrebné mať nainštalované:

* **Git**
* **Docker**
* Webový prehliadač (Chrome)

---

## Inštalácia a spustenie aplikácie

### 1. Stiahnutie projektu

```bash
git clone <GitHub_Repository_URL>
```

---

### 2. Spustenie aplikácie pomocou Docker Compose

V projekte sa nachádza súbor `docker-compose.yml`.
Po jeho spustení sa:

* spustia všetky potrebné kontajnery,
* automaticky vytvorí databáza a databázové tabuľky.

---

### 3. Otvorenie aplikácie

Aplikácia je dostupná na URL:

```
http://localhost
```

Aplikácia by mala byť pripravená na používanie.
